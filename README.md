# Como usar o cronometro?
 Utilize a tag script para chamar nosso script JS para incorporar ao site HTML
```html
<script src="cronometro.js"></script>
```

Comece chamando uma das funções abaixo para executar uma ação.

### Iniciar o contador

```javascript
start();
```

### Pausar o contador

```javascript
pause();
```


### Parar o contador

```javascript
stop();
```
